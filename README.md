# Car Doc

Car Doc is a serverless, cloud-first auto repair shop application built with
Next.js, TypeScript, Tailwind CSS, and the App Router.

## Development

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run lint
npm run build
```

The current project contains framework scaffolding and a generic landing page
only. Authentication, persistence, and legacy data import are intentionally not
included yet.
